# Data Protection

## At Rest

- Use database access controls.
- Optional disk encryption.
- Define retention windows and deletion routines for dormant accounts.

## In Transit

- TLS termination at edge or proxy.
- HSTS and secure headers where applicable.

## Privacy Controls (Japan)

- Publish a privacy notice aligned with APPI + PPC guidance.
- Disclose cross-border data transfers and safeguards when applicable.
- Provide contact path for access, correction, and deletion requests.
