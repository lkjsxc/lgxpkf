# Code Style

## Principles

- Functional programming with pure functions.
- Immutable domain structs.
- Small, focused modules.
- Web UI logic is written in TypeScript with explicit types.
- No JavaScript source files in the repository.

## Rust Guidelines

- Prefer iterators over loops.
- Avoid global mutable state.
- Keep file length under 200 lines.

## TypeScript Guidelines

- Favor pure functions and typed data shapes.
- Avoid mutable shared state and side effects outside initialization.
- Keep file length under 200 lines.
